1. Binary Search Tree Big-O

What's the big-O of inserting an element in an (unbalanced) binary search tree? What's the best-case big-O?

2. Binary Search Tree Average Height

What's the best-case big-O for the height of an (unbalanced) binary search tree? What's the average-case big-O (the average height if elements are inserted in a random order)?

3. Memoize

Memoize this function:

// Returns the number of corners in the nth approximation of the  Hilbert Curve.
long num_hilbert_curve_corners(int n) {
  if (n == 0) {
    return 0;
  }
  return 4 * num_hilbert_curve_corners(n - 1) + (n % 2 == 0 ? 4 : 2);
}

4. No Capitals

Write a function that takes a std::set<std::string>* and removes all strings that contain a capital letter, in-place (so the return type is void).

5. Counting Chars

Write a function that takes an std::ifstream& as input and returns an std::map, mapping from each character in the input to the number of times that character appeared in the input.

6. Subtrees

How many subtrees are there in a binary search tree with N nodes?

7. Segfault

Name two different causes of segmentation faults.
