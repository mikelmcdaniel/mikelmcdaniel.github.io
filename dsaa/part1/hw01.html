Homework 01. Text Editor

Goal of this assignment

The goal of this assignment is to practice your skills and test your knowledge about vectors and linked lists.

Once you've mastered vectors and linked lists, you should...

  • be able to implement a vector and linked list (including unit tests)
  • know the big-O of time and space for vectors and linked lists
  • know which data structure is better in theory (big-O)
  • be able to measure which data structure is better in practice
  • know how to use iterators

Problem

You and your friend are designing a basic text editor that you'd like to be fast. Your friend is writing the user interface and you're implementing some of the backend that actually edits the text.

To do this, you need to efficiently support a few operations:

  • move the cursor up/down (to the previous/next line)
  • move the cursor left/right (to the previous/next character, going to the previous/next line if needed)
  • add a character (just after the cursor)
    • if the character is '\n', then the current line should be split into two lines
  • remove a character (just before the cursor)
    • if the cursor is at the beginning of a line, then that line should be merged with the previous line
  • delete the line at the cursor

In other words, you need to support this interface:

class TextLines {
public:
  void move_cursor_up();
  void move_cursor_down();
  void move_cursor_left();
  void move_cursor_right();
  void add_char_at_cursor(char c);
  void remove_char_at_cursor();
  void delete_line_at_cursor();

  // These methods will be used for testing
  int cursor_line_number();
  int cursor_column_number();
  int line_length_at_cursor();
  string line_at_cursor();
};

Since this is a homework about linked lists and vectors, TextLines should be represented as a vector of vectors of chars, vector of linked lists of chars, linked list of vectors of chars, xor a linked list of linked lists of chars.
