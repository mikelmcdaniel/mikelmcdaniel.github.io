Homework 01. Compromise Search

The goal of this homework is to write an algorithm that gets the best of Linear Search and Binary Search by using Binary Search on large inputs until you narrow down the range to some 'small size', then using Linear Search for the remaining elements. -- Make sure your implementation switches from a Binary Search approach to a Linear Search approach!

The goal of this part of the assignment is to practice your skills translating an idea/algorithm to code.

//   Binary Search between low and high until the gap between low and high is <= small_size,

RandomAccessIt is a RandomAccessIterator, like std::vector<string>::begin() or std::vector<string>::end().

You can do many things with RandomAccessIt that you can do with pointers. For example, if you have two random access iterators (for the same collection), p1 and p2, you can...

  • dereference: *p1
  • increment (or decrement): ++p1 or p1++ (or --p1 or p1--)
  • compare: p1 == p2 or p1 < p2
  • subtract: p2 - p1
  • add an offset: p1 + some_int (e.g. p1 + 5)

You can use std::find as long as you only call it once and only on a range of elements that's <= small_size in size. (So you can't call it on the whole input)

If you can find some way to have std::binary_search switch to a Linear Search part-way through (the moment you've narrowed the search down to <= small_size elements), then you can use it but it's probably easier to just implement your own Binary Search that switches to Linear Search part-way through.

Simple Manual Test program

This program is provided to help you understand how compromise_search should work.

test_compromise_search.cpp 

#include <cmath>
#include <iostream>
#include <string>
#include <vector>

#include "compromise_search.h"

using namespace std;

int main() {
  vector<string> words = {"apple", "banana", "mango", "tangerine", "zuchini"};
  vector<string> words_to_search = {
      "apple", "mango", "zuchini", "aaa", "bbb", "yyy", "zzz"};
  for (const string& w : words_to_search) {
    for (int small_size = 0; small_size <= words.size(); ++small_size) {
      auto word_iterator = compromise_search(words.begin(), words.end(), w, small_size);
      cout
          << w << " was"
          << (word_iterator == words.end() ? " not" : "")
          << " found" << endl;
    }
    cout << endl;
  }
  cout << endl;
  return 0;
}
