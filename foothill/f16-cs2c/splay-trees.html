Splay Trees

General Idea

Splay Trees are a type of Balanced Binary Search Tree that move the last accessed element to the root of the tree. Searching, inserting, and removing all happen in exactly the same way as with Binary Search Trees, except that the node is splayed after (or in the case of removing before) that operation is done. A splay operation moves a node from wherever it is to the root of the tree by performing a bunch of tree rotations like the ones used in AVL trees.

Splaying

Searching

Search it like any Binary Search Tree, then splay the node you were searching for.

Inserting

Insert the node like any Binary Search Tree, then splay the inserted node.

Removing

Find the node to remove, splay it, then remove it just like you would in a Binary Search Tree.
