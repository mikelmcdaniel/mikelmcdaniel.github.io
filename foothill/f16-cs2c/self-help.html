Forums: How to ask questions

When you have a problem, a lot of people genuinely want to help you learn how to solve your problem. If you haven't read it already, please read "How to ask a good question?" by Stack Overflow so you can help others help you.

Your URL Bar (a.k.a. Google or Bing)

Never underestimate the power of literally typing in your question in your browser's URL bar.

If you don't how an algorithm works, like bubble sort, search for it!

If you try to compile some code and get an error message, search for it! (Copy and past the error message.)

C++

cppreference.com is a great C++ reference, but it is very technical and is best used for exploring things you've already studied, like std::vector::push_back.

Programming Practice

If you're looking for general programming practice, there are a lot of free sites out there. Most of the easy-to-use ones don't require you to submit code but ask you to write a program to calculate something, then enter your answer. Here are a few I recommend:

Site Focus
Coder Byte general algorthms and datastructures problems
Hacker Rank a wide variety of problems, sponsored by companies
Project Euler very math (number theory) intensive, starting out easy and getting much harder
Rosaland focused on bioinformatics, where biology meets computer science
